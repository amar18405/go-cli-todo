# Go Todo API

A full-stack backend API built with:

- Go (Golang)
- Gin Web Framework
- PostgreSQL

## Features

- RESTful API (CRUD)
- PostgreSQL integration
- JSON-based communication

## Endpoints

GET /todos  
POST /todos  
PUT /todos/:id  
DELETE /todos/:id  

## Tech Stack

- Go
- Gin
- PostgreSQL

go run .
